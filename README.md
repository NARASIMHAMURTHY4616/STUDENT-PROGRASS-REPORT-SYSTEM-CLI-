# 🎓 Student Progress Tracking System

A simple and efficient **Python-based console application** to manage student records, calculate marks, assign grades, generate remarks, and create structured progress reports.

---

## 🚀 Features

### 👨‍🎓 Student Management
- Add student details (Name, Roll No, Branch)
- Store data using Python dictionary
- Supports multiple students

---

### 📊 Marks Processing
- Input marks for multiple subjects
- Automatically calculates:
  - Average
  - Percentage
  - Maximum & Minimum marks

---

### 🏆 Grade System

| Average Marks | Grade |
|--------------|------|
| 90 – 100     | A    |
| 80 – 89      | B    |
| 70 – 79      | C    |
| 60 – 69      | D    |
| Below 60     | FAIL |

---

### 📌 Result Status
- **PASS** → All subjects ≥ 35  
- **FAIL** → Any subject < 35  

---

### 💬 Remarks Generator
- Generates feedback based on performance
- Examples:
  - Excellent performance! Keep it up.
  - Good job! You can do even better.
  - Average performance. Improve your weak subjects.
  - You failed in one or more subjects. Focus more and improve.

---

### ✏️ Modify Marks
- Update marks anytime
- Automatically recalculates:
  - Grade
  - Status
  - Percentage
  - Remarks

---

### 📄 Progress Report
Displays:
- Student Details
- Marks Summary
- Grade & Status
- Remarks
- Subject-wise marks
